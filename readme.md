# VSCode 可视化快捷命令按钮

方便的像一个IDE一样

## Features

当项目为一个npm项目并且有package.json的script以下命令时，可以快捷的关联四个按钮到
* npm start
* npm run build
* npm run publish
* npm run build:android


![scripts](images/banner.png)





> Tip: 你需要自行创建这四个npm script
